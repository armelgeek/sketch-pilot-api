**Données typiques SaaS** :
- 40% utilisent <25% de leur quota
- 35% utilisent 25-75%

- Starter : $12.30 × 40% = **$4.92** (vs $9 → **+$4 profit**)
- Creator : $24.60 × 45% = **$11.07** (vs $19 → **+$8 profit**)

## 🏛️ Impact Fiscal & Net-Net

Pour passer du profit brut au **Profit Net-Net** (ce qui reste réellement dans la poche).

### 1️⃣ Frais de Transaction (Stripe)
*Modèle : 2.9% + $0.30 par transaction*
- **Starter ($9)** : $0.26 + $0.30 = **$0.56** (6.2%)
- **Creator ($19)** : $0.55 + $0.30 = **$0.85** (4.5%)
- **Pro ($49)** : $1.42 + $0.30 = **$1.72** (3.5%)

### 2️⃣ Taxes & Impôts (Modèle France/EU)
*Hypothèse : Vente B2C (TVA 20%) ou B2B (Auto-liquidation).*

| Poste | Calcul | Impact / $100 |
|-------|--------|---------------|
| **TVA (20%)** | $100 / 1.2 | -$16.67 |
| **Frais Stripe** | Moyenne 4% | -$4.00 |
| **Impôt (IS/IR)** | ~25% sur résultat | Variable |

### 3️⃣ Calcul du Profit Net-Net (Exemple Creator $19)

| Étape | Montant | Reste |
|-------|---------|-------|
| **Prix Payé (TTC)** | **$19.00** | $19.00 |
| TVA (20%) | -$3.17 | $15.83 |
| Frais Stripe | -$0.85 | $14.98 |
| Coût API (Usage moyen) | -$11.07 | $3.91 |
| **Profit Brut Net de TVA** | | **$3.91** |
| IS / Charges (25%) | -$0.98 | **$2.93** |
| **Profit NET-NET** | | **~15% du CA** |

> [!IMPORTANT]
> Avec un pricing agressif, la marge nette réelle se situe entre **15% et 20%**. Le levier principal de croissance est le passage en B2B (récupération TVA) et l'optimisation des coûts API par le cache.
