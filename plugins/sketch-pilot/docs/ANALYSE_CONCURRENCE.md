## 🚨 Menaces concurrentielles

### Menace 1 : Synthesia baisse ses prix

**Probabilité** : Faible (25%)  
**Impact** : Élevé  

**Scenario** :
- Synthesia lance un plan "Indie" à $15/mois
- Cible les créateurs solo
- Utilise leur brand recognition

**Mitigation** :
- Notre différence de prix reste significative ($9 vs $15)
- Notre simplicité reste supérieure
- Pivot rapide vers B2B si nécessaire

---

### Menace 2 : Nouveau concurrent low-cost

**Probabilité** : Élevée (70%)  
**Impact** : Moyen  

**Scenario** :
- Startup lance outil similaire à même prix
- Marketing agressif
- Features additionnelles

**Mitigation** :
- First-mover advantage (build community fast)
- Amélioration continue features
- Lock-in via intégrations
- Brand building rapide

---

### Menace 3 : OpenAI/Google lance outil grand public

**Probabilité** : Moyenne (50%)  
**Impact** : Très élevé  

**Scenario** :
- Sora (OpenAI) devient public et pas cher
- Google lance Veo public avec interface
- Prix compétitif grâce à économie d'échelle

**Mitigation** :
- Pivot vers "sur-couche à valeur ajoutée"
- Focus sur use cases spécifiques (éducation, etc.)
- Intégration avec leur APIs
- Différenciation par templates/styles

---

### Menace 4 : Consolidation du marché

**Probabilité** : Élevée (60%)  
**Impact** : Moyen  

**Scenario** :
- Adobe/Canva/Figma achètent concurrents
- Intégration dans suites existantes
- Pricing bundlé attractif

**Mitigation** :
- Positionner pour acquisition (exit strategy)
- Construire valeur unique (data, community)
- Focus niche où géants ne vont pas

1. Brand recognition faible

### Recommandations stratégiques

#### 1. **Embrasser le positionnement low-cost**
- Ne pas essayer de rivaliser avec Synthesia sur qualité
- Assumer le style stickman comme différenciateur
- Marketing basé sur accessibilité et simplicité

#### 2. **Acquisition agressive early-stage**
- Profiter de l'avantage prix pour capturer volume
- Objectif : 10,000 users Année 1
- Community building pour defensibility

#### 3. **Monitor concurrence établie**
- Veille active sur Synthesia, Pictory, Descript
- Réaction rapide si baisse de prix
- Identifier opportunités de collaboration/intégration

#### 4. **Préparer pivot si nécessaire**
- Si géants entrent sur notre segment : pivot B2B ou niche
- Si prix API explosent : pivot vers freemium avec limitations
- Rester agile, ne pas s'enfermer

#### 5. **Focus sur use cases spécifiques**
- Éducation en ligne
- Social media content
- Marketing vidéo volume
- Ne pas essayer d'être tout pour tous
